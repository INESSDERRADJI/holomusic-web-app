# HOLOMUSIC
> This repository presents a portfolio overview of the project. The full source code is not publicly available.

Music-oriented web application developed as an academic web and database project.

## Overview

HOLOMUSIC is a web application designed to present music-related content through structured pages and database-driven features.

The project was built and deployed on OVH, with MySQL database integration and external libraries for interface and user experience improvements.

## Main Features

- Article publication and reading
- Rich text editor for article creation
- User registration, login and profile management
- Comments and replies
- Like/dislike interactions
- Dynamic article search with AJAX
- PDF export for articles
- Responsive interface

## Technologies

- PHP
- MySQL
- HTML/CSS
- JavaScript
- Bootstrap / MDBootstrap
- TinyMCE
- Dompdf
- MAMP
